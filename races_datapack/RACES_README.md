# Minecraft Races Datapack

A comprehensive Minecraft Purpur server datapack with 5 unique playable races featuring:
- Mandatory GUI race selection
- LuckPerms integration with suffixes
- Custom race abilities and effects
- DeluxeMenus GUI interface

## Races

1. **Apisapien** - Slow falling + bee immunity
2. **Dwarf** - Night vision + sun damage resistance
3. **Elf** - +20% attack speed, -5% damage
4. **Undead** - Sun damage, night regeneration
5. **Ogre** - +20% damage, -25% attack speed

## Requirements

- Purpur server
- LuckPerms, DeluxeMenus, Vault, TAB, SimpleScore plugins
